# Bhaskar Neogi — Dark Interactive Portfolio

A dark, interactive Next.js portfolio generated from Bhaskar Neogi's CV.

## Tech

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS 3
- Pure CSS + React interactive animations

## Features

- Dark aurora/cyber theme
- Mouse spotlight animation
- Animated orbit hero
- Interactive project explorer
- Hover-based skill matrix
- Career timeline
- Download CV button

## Run locally

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

## Customize

Most portfolio data is inside `app/page.tsx` in the `skills`, `projects`, and `experience` arrays.
The CV file is available at `public/bhaskar-neogi-cv.pdf`.
